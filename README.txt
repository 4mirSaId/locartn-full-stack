LoCarTn is a modern car rental platform connecting customers with rental agencies across Tunisia. The platform features user authentication, car listings management, and reservation systems.

##  Features

- **User Roles System**:
  - Customers browse and rent cars
  - Agencies manage their car fleet
  - Admin dashboard for platform management

- **Car Management**:
  - Add/edit/delete car listings
  - Image uploads with Cloudinary
  - Advanced search and filters

- **Booking System**:
  - Real-time availability check
  - Reservation management
  - Rental history

- **Modern UI**:
  - Responsive Tailwind CSS design
  - Dark/light mode support
  - Interactive components
